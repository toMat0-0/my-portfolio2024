Datasets

The following materials were used in this sound-based dog/cat emotion classification project.

Datasets

  • Dog Vocalizations Dataset – Download
    Includes 45 barks, 26 grunts, and 23 growls in .wav format with emotion annotations.
  • Cat Vocalizations Dataset – Download
    Includes ~400 vocalizations used for secondary testing and model comparison.

Code Files

  • Python Script (Feature Extraction) – Download
    Used to convert .wav files into frequency features and generate spectrograms.
  • R Script (Visualization & Modeling) – Download
    Contains PCA, LDA, and 3D visualization with Plotly.
